# PowerShell Automation: New-VM.ps1
Author: Sana Rehman  
Date: 2025-11-10  

## Purpose
Automates creation of a new Hyper-V virtual machine with parameters for
name, memory, disk size, network switch, and storage path.

## How to Run
Run the script in PowerShell ISE (as Administrator) and follow prompts.

## Output
Displays a confirmation message and VM details once created.
